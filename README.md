# Faleh_enala

الصفحة التعريفية الرسمية لـ **فالح عبدالله الفالح** — المؤسس والرئيس التنفيذي لمجموعة إناله للفنادق والمنتجعات.

Official profile website for **Faleh Abdullah Al Falih** — Founder & CEO of Enala Hotels & Resorts.

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS v4
- Framer Motion

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
